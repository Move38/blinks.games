Welcome to Blinks @ Global Game Jam 2020!

We’re glad you’ve decided to jam with us this weekend! Here’s what you’ll need to get started

1. Join the Slack

Join our dedicated Slack channel to have access to expert advice and community discussions with developers at the site and all over the world!


2. Install Arduino




Make sure you download the Arduino IDE 1.8.11, not the online dev tools.


3. Get the Blinks SDK

After you have installed Arduino and launched the application, you will notice that Arduino has created a folder in your Documents folder named Arduino. Download the zip file above, then follow these steps to install the Blinks Library:

  1. Unzip the Blinks Library you downloaded into ~/Documents/Arduino/hardware. Note: If this is your first time with custom hardware, you will need to create the hardware folder.
  2. Restart Arduino, then click `Tools` in your menu bar, and navigate down to Board: you should now see Move38: Blinks Tile at the bottom. Select it.
  3. Click on Tools in your menu bar, and navigate down to Programmer: select the programmer Blinks USBTinyISP
